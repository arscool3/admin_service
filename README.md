**Сервис для администрирования**
-----

**Установка зависимотей**

`pip install requirements.txt`

**Запуск базы**

`docker-compose up`

**Заполнение базы данными**

`python manage.py create_sample_data`


**API**
------

**Просмотр списка клиентов**
----

* **URL**

  /api/clients/

* **Method:**

  `GET`
  
*  **Параметры**

   **Обязательно:**
 
   `NONE`


* **Ответ**

  * **Code:** 200 <br />
    **Content:** `cписок всех клиентов`
